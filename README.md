# 🏏 IPL Cricket Analytics Dashboard

An interactive **Python + Streamlit** web application for analyzing IPL player performance, team statistics, match outcomes, and cricket insights using dynamic visualizations.

---

## 📌 Project Overview

The IPL Cricket Analytics Dashboard is a data analytics project developed using **Python**, **Pandas**, **NumPy**, **Plotly**, and **Streamlit**. It transforms raw IPL player performance data into an interactive dashboard that helps users explore batting, bowling, and team statistics.

This project demonstrates practical skills in:

- Python Programming
- Data Analysis
- Data Visualization
- Dashboard Development
- Interactive Web Applications

## 🚀 Installation

### 1. Clone the Repository

```bash
git clone https://github.com/yourusername/IPL-Cricket-Analytics-Dashboard.git
```

### 2. Navigate to the Project

```bash
cd IPL-Cricket-Analytics-Dashboard
```

### 3. Install Dependencies

```bash
pip install -r requirements.txt
```

### 4. Run the Application

```bash
streamlit run src/dashboard.py
```
